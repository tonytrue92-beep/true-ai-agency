05 Деплой

GitHub Pages — бесплатный хостинг за 3 команды

Репозиторий создан, Pages включён через API. Через ~2 минуты сайт был доступен по URL.

# Создаём репозиторий и пушим gh repo create true-ai-agency --public --source=. --push # Включаем GitHub Pages через API gh api repos/tonytrue92-beep/true-ai-agency/pages \ --method POST \ --field 'source[branch]=main' \ --field 'source[path]=/' # Через ~2 минуты: https://tonytrue92-beep.github.io/true-ai-agency/ → 200 OK
1

git init + commit

Один HTML-файл, один коммит.

2

gh repo create

Публичный репо создан и запушен через GitHub CLI.

3

Pages API

GitHub Pages включён программно — без ручной настройки в браузере.

4

Проверка 200 OK

curl вернул 200 — сайт живёт.

    Ресёрч → стратегия → продукт → код → деплой. Без переключения между приложениями, фрилансерами и платформами.
