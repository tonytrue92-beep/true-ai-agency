# От "проанализируй рынок" до готового сайта — за одну сессию

**Кейс: как мы запустили TRUE AI AGENCY с нуля за несколько часов**

---

## Контекст

Предприниматель (Антон) + AI-агент (Claude / OpenClaw).  
Без дизайнера. Без разработчика. Без агентства.  
Один запрос запустил цепочку из 5 шагов — и на выходе живой сайт.

---

## Шаг 1 — Ресёрч рынка

**Запрос:** _"пройди и собери информацию по разным источникам где сейчас модно применять агентов на базе openclaw на российском рынке"_

**Что сделал агент:**

- Открыл браузер, прошёлся по Яндекс-выдаче
- Зашёл на wikibot.pro, generation-ai.ru, botech.pro, vc.ru/ai
- Собрал реальные кейсы 2025 года: Smartavia, WinWork, Займер, Skillfactory, КНАУФ, Совкомбанк, Альфа-банк
- Проанализировал рекламные объявления конкурентов — что продают и как

**Что выяснили:**

| Ниша | Спрос | Кейс |
|------|-------|------|
| Клиентская поддержка | 🔥🔥🔥 | Smartavia: 75% без оператора |
| Финансы/взыскание | 🔥🔥 | Займер, Альфа-банк |
| Онлайн-образование | 🔥🔥 | Skillfactory: 40% без оператора |
| B2B/производство (RAG) | 🔥🔥 | КНАУФ, Совкомбанк |
| Telegram-боты МСБ | 🔥 (перегрет) | Много дешёвых игроков |

**Главный вывод:**  
Рынок продаёт "ИИ вместо менеджера". Между дешёвыми ботами (Botech) и тяжёлым enterprise (Wikibot) — огромная дыра для умных агентов МСБ.

## Шаг 2 — Позиционирование

На основе ресёрча сформулировали главное отличие:

> **Не "мы делаем ботов"**  
> → **"Мы строим AI-команды которые работают вместо сотрудников"**

Это другой разговор, другой чек, другой клиент.

## Шаг 3 — Продуктовая линейка по модели Apple

**Запрос:** _"давай сделаем пакеты услуг по примеру Apple — человек покупает основу и дальше мы делаем дополнительные продажи"_

**Логика модели:**

```
iPhone = AI-Сотрудник (основной продукт)
AirPods = Долгая память (клиент сам замечает что не хватает)
Lightning = CRM-интеграция (без этого данные теряются)
iCloud = База знаний (агент отвечает неточно → очевидная потребность)
AppleCare = Техподдержка (подписка, работает пассивно)
Apple Watch = Второй агент (другая роль в команде)
```

**Структура продукта:**

**Основа — AI-Сотрудник** (80–150К₽)  
Роли: Менеджер / Куратор / Поддержка / Контент-ассистент

**Tier 1 — без них вполсилы** (продаются в первый месяц):
- 🧠 Долгая память — от 30К₽
- 🔗 CRM-интеграция — от 40К₽  
- 📚 База знаний (RAG) — от 50К₽
- 🔧 Техподдержка — от 15К₽/мес

**Tier 2 — расширение экосистемы** (через 1–3 месяца):
- 👥 Второй агент — скидка 30%
- 📱 Новый канал (WhatsApp/сайт) — от 25К₽
- 📊 Аналитика — от 10К₽/мес
- 🎙️ Голос — от 20К₽

**Пакеты:**
- Старт: от 80К₽
- Стандарт: от 170К₽ + 15К/мес ⭐
- AI-Команда: от 350К₽

**Средний чек при апсейле:** 170–380К₽ разово

## Шаг 4 — Создание лендинга

**Запрос:** _"давай в лендинг"_ + ответы на 5 вопросов:
1. Название: TRUE AI AGENCY
2. Формат: готовый HTML-сайт
3. Кейсы: пока не нужны
4. CTA: форма, данные передаются в Telegram
5. Лицо: Антон Поляков (@tonytruee)

**Что сделал агент:**

1. Написал 700 строк HTML/CSS/JS — один файл, без зависимостей
2. Тёмный дизайн в стиле Apple (чёрный фон, золотые акценты)
3. Блоки: Hero → Проблема → Продукт → Аксессуары → Путь клиента → Пакеты → Об Антоне → Форма
4. Форма при сабмите отправляет заявку прямо в Telegram (@tonytruee) через Bot API
5. Адаптив под мобилку

## Шаг 5 — Деплой на GitHub Pages

```bash
# Создали репозиторий
gh repo create true-ai-agency --public --source=. --push

# Включили GitHub Pages через API
gh api repos/tonytrue92-beep/true-ai-agency/pages \
  --method POST \
  --field 'source[branch]=main' \
  --field 'source[path]=/'
```

**Через ~2 минуты сайт был живым.**
